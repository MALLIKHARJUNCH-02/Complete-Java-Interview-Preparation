# DSA/19-Searching.md

## Definition

Searching is the process of finding a specific element in a collection of data.


---

# Must-Know Searching Algorithms

| Algorithm            | Time Complexity      |
| -------------------- | -------------------- |
| Linear Search        | O(n)                 |
| Binary Search        | O(log n)             |
| Jump Search          | O(√n)                |
| Exponential Search   | O(log n)             |
| Interpolation Search | O(log log n) Average |
| BFS                  | O(V + E)             |
| DFS                  | O(V + E)             |
| A* Search            | Heuristic Based      |
