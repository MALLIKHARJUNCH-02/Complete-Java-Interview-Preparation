# DSA/20-Sorting.md

## Definition

Sorting is the process of arranging data in ascending or descending order.



# Must-Know Sorting Algorithms

| Algorithm      | Time Complexity    |
| -------------- | ------------------ |
| Bubble Sort    | O(n²)              |
| Selection Sort | O(n²)              |
| Insertion Sort | O(n²)              |
| Merge Sort     | O(n log n)         |
| Quick Sort     | O(n log n) Average |
| Heap Sort      | O(n log n)         |
| Counting Sort  | O(n + k)           |
| Radix Sort     | O(d × (n + k))     |
| Bucket Sort    | O(n + k)           |
