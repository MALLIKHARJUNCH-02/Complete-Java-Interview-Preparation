# DSA/08-BST.md

## Definition

A Binary Search Tree (BST) is a binary tree where:

* Left subtree contains values smaller than the root.
* Right subtree contains values greater than the root.
* Both left and right subtrees are also BSTs.

Time Complexity:

* Search: O(log n)
* Insert: O(log n)
* Delete: O(log n)
