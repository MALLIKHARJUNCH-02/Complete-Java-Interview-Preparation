# DSA/09-Heap.md

## Definition

A Heap is a complete binary tree that satisfies the Heap Property.

Types:

* Max Heap → Parent is greater than children.
* Min Heap → Parent is smaller than children.

Applications:

* Priority Queue
* Scheduling
* Dijkstra's Algorithm
* Heap Sort
