# DSA/10-Graph.md

## Definition

A Graph is a non-linear data structure consisting of:

* Vertices (Nodes)
* Edges (Connections)

Types:

* Directed Graph
* Undirected Graph
* Weighted Graph
* Unweighted Graph
* Cyclic Graph
* Acyclic Graph

Representations:

* Adjacency Matrix
* Adjacency List
