# 16-WindowFunctions.md

## Definition

Window Functions perform calculations across a set of rows related to the current row without collapsing the result set.

Most Important Functions:

* ROW_NUMBER()
* RANK()
* DENSE_RANK()
* NTILE()
* LEAD()
* LAG()
* FIRST_VALUE()
* LAST_VALUE()
* SUM() OVER()
* AVG() OVER()
