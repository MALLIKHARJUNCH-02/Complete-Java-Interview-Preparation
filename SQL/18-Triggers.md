# 18-Triggers.md

## Definition

A Trigger is a database object that automatically executes when a specified event occurs on a table.

Types:

* BEFORE INSERT
* AFTER INSERT
* BEFORE UPDATE
* AFTER UPDATE
* BEFORE DELETE
* AFTER DELETE
