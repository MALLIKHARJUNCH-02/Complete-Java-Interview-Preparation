# 22-Multithreading.md

## Definition

Multithreading is a Java feature that allows concurrent execution of two or more threads for maximum CPU utilization.

Topics Covered:

* Thread Class
* Runnable Interface
* Thread Lifecycle
* Synchronization
* Inter-Thread Communication
* Deadlock
* Executor Framework
* Callable & Future
* Concurrent Collections
* Fork Join Framework
